What's its Function?

It's Old School!

This mini PC was fully 3D Printed & Assembled. It runs primarily off a Raspberry Pi 4 Model B (4GB) and currently has OctoPrint Installed on a Micro-SD Card. This software allows the user to control their 3D Printer on either a LAN or WAN connection.

Time to Build!
The 3D Design is listed on Printables created by fantasticmrdavid. Once all parts are printed they are required to be assembled with multiple KM Flat Head Philips Screws.
Printing the multiple parts was done on a Creality Ender 3 V2 Neo. Complemented With CURA Slicing Software to adjust printing parameters and advanced settings.
The Electronics I decided to mount into the PC Case was a Raspberry Pi 4 and a 3.5 Inch 1080P HD Screen (MPI3508). Necessary cables include a Ribbon Cable & Micro-HDMI to HDMI Converter.
The Raspberry Pi is currently running the latest iteration OctoPrint Software (v 1.9.3). With this software I am able to view my printer's settings (Temperatures, Axis, Printing Speeds). Alongside this I'm able to view prints real-time from my smartphone/electronic devices.
